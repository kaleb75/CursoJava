Bienvenido al Curso de Java

Este curso está diseñado para que aprendas Java desde cero. Incluye ejemplos prácticos y explicaciones detalladas para que puedas empezar a programar.

Conceptos Básicos

En esta sección aprenderás los fundamentos de Java, como la estructura básica de un programa, variables, tipos de datos y operadores.

Estructura básica de un programa Java

        
/*
 * Este es un programa básico en Java.
 * Siempre comienza con la definición de una clase pública.
 */
public class HolaMundo {

    // El método main es el punto de entrada del programa.
    public static void main(String[] args) {
        // Este comando imprime un mensaje en la consola.
        System.out.println("¡Hola, Mundo!");
    }
}

Explicación del código:

  • public class HolaMundo: Define una clase llamada HolaMundo. En Java, todo el código debe estar dentro de una clase.
  • public static void main(String[] args): Este es el método principal donde comienza la ejecución del programa.
  • System.out.println("¡Hola, Mundo!");: Imprime el texto "¡Hola, Mundo!" en la consola.

Estructuras de Control

En Java, puedes controlar el flujo de tu programa con estructuras como condicionales y bucles.

Ejemplo de condicional if

        
/*
 * Este programa evalúa si un número es mayor que 10.
 */
public class EjemploIf {
    public static void main(String[] args) {
        int numero = 15; // Declaración de una variable de tipo entero

        // Condicional if para verificar si el número es mayor que 10
        if (numero > 10) {
            System.out.println("El número es mayor que 10.");
        } else {
            System.out.println("El número no es mayor que 10.");
        }
    }
}

Explicación del código:

  • int numero = 15;: Declara una variable llamada numero y le asigna el valor 15.
  • if (numero > 10): Verifica si el valor de numero es mayor que 10.
  • System.out.println(...): Imprime un mensaje según el resultado de la condición.

Programación Orientada a Objetos (POO)

La Programación Orientada a Objetos (POO) es un paradigma que organiza el código en **clases** y **objetos**, permitiendo modelar problemas del mundo real de forma más intuitiva.

  • Conceptos Clave de POO
  • Clases y Objetos
  • Herencia
  • Polimorfismo
  • Interfaces

Conceptos Clave de POO

POO se basa en cuatro principios fundamentales:

  1. Clases y Objetos: Una clase es un plano o plantilla que define las características y comportamientos de un objeto. Un objeto es una instancia concreta de esa clase.
  2. Encapsulación: Consiste en ocultar los detalles internos de una clase y exponer únicamente lo necesario. Esto se logra utilizando modificadores de acceso como private, public, y protected.
  3. Herencia: Permite que una clase (hija) adquiera propiedades y métodos de otra clase (padre). Esto fomenta la reutilización de código.
  4. Polimorfismo: La capacidad de un método o clase de comportarse de diferentes maneras dependiendo del contexto.

Clases y Objetos

Las clases y los objetos son los pilares fundamentales de POO. Una clase define un conjunto de atributos (datos) y métodos (acciones), mientras que un objeto es una instancia que utiliza esos atributos y métodos.

                
      /*
       * Clase que representa a una persona.
       */
      public class Persona {
          // Atributos de la clase
          String nombre;
          int edad;
      
          // Método para mostrar información
          public void mostrarInfo() {
              System.out.println("Nombre: " + nombre);
              System.out.println("Edad: " + edad);
          }
      }
      
      public class Main {
          public static void main(String[] args) {
              // Creación de un objeto de la clase Persona
              Persona persona1 = new Persona();
              
              // Asignación de valores a los atributos
              persona1.nombre = "Juan";
              persona1.edad = 30;
      
              // Llamada al método de la clase
              persona1.mostrarInfo();
          }
      }

Explicación:

  • public class Persona: Define una clase llamada Persona con atributos y métodos.
  • Persona persona1 = new Persona();: Crea un objeto llamado persona1 de tipo Persona.
  • persona1.nombre = "Juan";: Asigna el valor "Juan" al atributo nombre del objeto.
  • persona1.mostrarInfo();: Llama al método para imprimir la información de la persona.

Herencia

La herencia permite que una clase reutilice atributos y métodos de otra clase. Esto fomenta la reutilización de código y reduce redundancias.

                
      /*
       * Clase base o superclase Animal.
       */
      public class Animal {
          String nombre;
      
          public void hacerSonido() {
              System.out.println("El animal hace un sonido.");
          }
      }
      
      /*
       * Clase Perro que hereda de Animal.
       */
      public class Perro extends Animal {
          public void hacerSonido() {
              System.out.println("El perro ladra.");
          }
      }
      
      public class Main {
          public static void main(String[] args) {
              // Crear un objeto de la clase Perro
              Perro miPerro = new Perro();
              miPerro.nombre = "Rex";
              miPerro.hacerSonido();  // Salida: El perro ladra.
          }
      }

Explicación:

  • public class Animal: Clase base que define un atributo y un método genérico.
  • public class Perro extends Animal: Clase que hereda de Animal y sobrescribe el método hacerSonido.
  • miPerro.hacerSonido(): Llama al método sobrescrito en la clase Perro.

Polimorfismo

El polimorfismo permite que un objeto adopte diferentes formas dependiendo del contexto. Por ejemplo, un método puede comportarse de manera distinta en función de la clase que lo implemente.

                
      /*
       * Clase base o superclase Figura.
       */
      public class Figura {
          public void dibujar() {
              System.out.println("Dibujando una figura genérica.");
          }
      }
      
      /*
       * Subclase para Círculo.
       */
      public class Circulo extends Figura {
          public void dibujar() {
              System.out.println("Dibujando un círculo.");
          }
      }
      
      /*
       * Subclase para Cuadrado.
       */
      public class Cuadrado extends Figura {
          public void dibujar() {
              System.out.println("Dibujando un cuadrado.");
          }
      }
      
      public class Main {
          public static void main(String[] args) {
              Figura figura1 = new Circulo();
              Figura figura2 = new Cuadrado();
      
              figura1.dibujar();  // Salida: Dibujando un círculo.
              figura2.dibujar();  // Salida: Dibujando un cuadrado.
          }
      }

Explicación:

  • Figura figura1 = new Circulo();: Declara un objeto de tipo Figura, pero asigna una instancia de Circulo.
  • figura1.dibujar();: Llama al método dibujar, que se comporta según la implementación en Circulo.

Proyectos: Java + Eclipse + MySQL

En esta sección aprenderás a desarrollar un proyecto completo combinando **Java**, **Eclipse IDE**, y **MySQL**. El objetivo es crear un **Sistema de Gestión de Usuarios** con operaciones CRUD (Crear, Leer, Actualizar, Eliminar).

¿Prefieres aprender con un video? Aquí tienes una guía visual que cubre muchos de estos pasos: Ver Video Tutorial.

1. Instalación y Configuración

Asegúrate de instalar las herramientas necesarias:

  • Java Development Kit (JDK): Descargar. Instala la versión más reciente y configura la variable de entorno JAVA_HOME.
  • Eclipse IDE: Descargar Eclipse. Usa la edición para desarrolladores Java.
  • MySQL Server y MySQL Workbench: Descargar. Configura un usuario y contraseña para el servidor.
  • MySQL Connector (JDBC): Descargar. Este archivo JAR es esencial para conectar Java con MySQL.

Consejo: Asegúrate de que tu servidor MySQL esté en ejecución antes de continuar.

2. Crear la Base de Datos

Usa el siguiente script SQL en **MySQL Workbench** para crear la base de datos y su estructura:

                  
              -- Crear la base de datos
              CREATE DATABASE sistema_usuarios;
              
              -- Seleccionar la base de datos
              USE sistema_usuarios;
              
              -- Crear la tabla usuarios
              CREATE TABLE usuarios (
                  id INT AUTO_INCREMENT PRIMARY KEY,
                  nombre VARCHAR(50) NOT NULL,
                  correo VARCHAR(100) NOT NULL,
                  edad INT NOT NULL
              );

Este script crea una base de datos llamada sistema_usuarios y una tabla usuarios con los campos necesarios.

3. Configurar el Proyecto en Eclipse

  1. Abre Eclipse y crea un proyecto Java: File > New > Java Project.
  2. Agrega el conector de MySQL (JDBC) al proyecto:
    1. Haz clic derecho en tu proyecto, selecciona Build Path > Add External Archives.
    2. Selecciona el archivo JAR del conector descargado previamente.
  3. Crea una nueva clase Java para manejar la conexión con la base de datos. Por ejemplo, SistemaUsuarios.

4. Código Java

Utiliza este código para conectar Java con MySQL y realizar operaciones CRUD:

                  
              import java.sql.*;
              
              public class SistemaUsuarios {
                  // Datos de conexión
                  private static final String URL = "jdbc:mysql://localhost:3306/sistema_usuarios";
                  private static final String USER = "root"; // Cambia por tu usuario
                  private static final String PASSWORD = ""; // Cambia por tu contraseña
              
                  public static void main(String[] args) {
                      try (Connection conn = DriverManager.getConnection(URL, USER, PASSWORD)) {
                          System.out.println("Conexión exitosa a la base de datos.");
              
                          // Insertar usuario
                          String insertQuery = "INSERT INTO usuarios (nombre, correo, edad) VALUES (?, ?, ?)";
                          try (PreparedStatement ps = conn.prepareStatement(insertQuery)) {
                              ps.setString(1, "Juan Pérez");
                              ps.setString(2, "juan.perez@mail.com");
                              ps.setInt(3, 28);
                              ps.executeUpdate();
                              System.out.println("Usuario insertado con éxito.");
                          }
              
                          // Leer usuarios
                          String selectQuery = "SELECT * FROM usuarios";
                          try (Statement stmt = conn.createStatement();
                               ResultSet rs = stmt.executeQuery(selectQuery)) {
                              System.out.println("Usuarios registrados:");
                              while (rs.next()) {
                                  System.out.printf("ID: %d, Nombre: %s, Correo: %s, Edad: %d%n",
                                          rs.getInt("id"), rs.getString("nombre"),
                                          rs.getString("correo"), rs.getInt("edad"));
                              }
                          }
              
                      } catch (SQLException e) {
                          e.printStackTrace();
                      }
                  }
              }

5. Explicación del Código

  • DriverManager.getConnection: Establece la conexión con la base de datos usando la URL, usuario y contraseña.
  • PreparedStatement: Ejecuta consultas parametrizadas, como insertar datos en la tabla usuarios.
  • Statement: Permite ejecutar consultas simples, como leer datos.
  • try-with-resources: Cierra automáticamente recursos como conexiones y declaraciones.

6. Mejoras Futuras

  • Agregar una interfaz gráfica con **JavaFX** o **Swing**.
  • Implementar más funciones, como editar y eliminar usuarios.
  • Validar los datos antes de interactuar con la base de datos.
  • Crear logs de actividad para auditar los cambios realizados en la base de datos.
  • Configurar un archivo de propiedades para manejar la conexión a la base de datos.
